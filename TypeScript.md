## Introduction
* Typescript add type to javascript
* Typescript is superset of javascript

### modules
* `typescript` module used to run ts file
* `tsc` is used to compile ts file to js file in terminal using typescript module
* `ts-node` is another module to direcly compile and execute ts file with one command.

### type
* a `type` is a label that describes the different properties and methods that a value has
* every value has a `type`.
* Two types of `type` are:
    1. Primitive type

    | type | description |
    |--------|------------|
    | string | text data |
    | number | numeric values |
    | boolean | true or false |
    | null | null |
    | undefined | undefined/uninitialized value |
    | symbol | unique constant value |

    2. Object Types - functions, arrays, classes, etc
